KMP算法步骤

  1. 求模式串每个子串前后缀最长公共元素长度（前缀不包括最后一个字符，后缀不包括最前面的字符

  2. 求next数组

  3. 匹配

模式串：a=ABCDABAC

子串 前缀 后缀 最长公共元素长度 i next数组 优化后的next数组
A 空 空 0 0 -1 -1
AB A B 0 1 0 0
ABC AB BC 0 2 0 0
ABCD ABC BCD 0 3 0 0
ABCDA ABCD BCDA 1 4 0 -1
ABCDAB ABCDA BCDAB 2 5 1 0
ABCDABA ABCDAB BCDABA 1 6 2 2
ABCDABAC ABCDABD BCDABAC 0 7 1 1

next数组

上面表格观察可得next数组是最长公共元素全向下移动一位，首位置为-1

已知next[0]到next[i]，求next[i+1]

next数组next[i]表示的是在i处失配

以ABCDABCD为例，求next[6]，和next[5]=1，如果a[next[5]]=a[6]，那么next[6]=next[5]+1所以next[6]=2

而a[next[6]]!=a[7]，则需要往前面的字串去递推(没看懂并大为震撼)
